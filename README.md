# Audvyn

Audvyn is a free student-built music education platform for classrooms. It helps students practice ear training, basic music theory, sight-reading, and staff-based music skills while giving teachers lightweight class, assignment, and results tools.

## Current beta features

- Ear Training Gym
  - Intervals
  - Chords
  - Cadences
  - Audio playback
  - Instant explanations

- Theory Tester
  - Notes
  - Key signatures
  - Written intervals
  - Triads
  - Roman numerals
  - Rhythm basics

- Sight-Reading Coach
  - Generated beginner melodies
  - Staff display
  - Starting pitch playback
  - Full melody playback
  - Note-name reveal

- Staff Lab
  - Teaching templates
  - Staff notation rendering
  - Playback
  - Basic note editing
  - Tempo control

- Classroom tools
  - Teacher and student accounts
  - Class join codes
  - Assignment creation
  - Student assignment runner
  - Saved attempts
  - Teacher results dashboard

## Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Vercel
- Tone.js
- VexFlow
